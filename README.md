# Smart Retail AI Engine 🛍️🤖

## Overview
An AI-powered Customer Segmentation and Recommendation System built with Python and Scikit-learn. This engine analyzes raw e-commerce transaction data to group customers based on their purchasing behavior and intelligently suggests new products they are likely to buy.

## Features
* **Data Engineering:** Cleaned, filtered, and transformed raw transactional datasets using Pandas.
* **RFM Analysis:** Calculated Recency, Frequency, and Monetary values to understand customer behavior.
* **Customer Segmentation (K-Means):** Clustered customers into 4 actionable business segments (e.g., VIP Whales, Everyday Customers, Lost Customers) using K-Means Clustering and the Elbow Method.
* **Recommender System (NMF):** Built a Collaborative Filtering engine using Non-negative Matrix Factorization (NMF) and Cosine Similarity to recommend personalized items.

## Tech Stack
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Machine Learning:** Scikit-learn (K-Means, NMF, StandardScaler)
* **Data Visualization:** Matplotlib
